3 Sparse stereo CUDA kernel (E) Later

1-2주 · Custom CUDA kernel for 6-keypoint depth

Goal: 6 keypoint (L/R hip/knee/ankle) 의 sparse stereo matching. Dense SGM 대체.

Sparse stereo algorithm:
  Input: L_image (GPU), R_image (GPU), 6 keypoint pixel (x_l, y_l)

  For each keypoint:
    1. Patch around (x_l, y_l) — 24×24 window in L_image
    2. Search in R_image along epipolar line
    3. Sub-pixel disparity refinement (parabolic fit)
    4. Z = fx × baseline / disparity
    5. σ_Z = Z² × σ_disparity / (fx × baseline)  ← per-kp uncertainty!

  Output: 6 × (X, Y, Z, σ_Z)

→ ~0.5ms total (6 × 80μs on GPU)
→ Paper claim 의 per-keypoint uncertainty (E 의 bonus)

6 C++ inference loop (B + A + C) Later

1-2주 · 전체 inference pipeline 의 C++ migration

Loop:

while (running) {
    // E: RawBuffer instead of dense depth
    zed.grab(runtime);
    zed.retrieveImage(left, sl::VIEW::LEFT_GRAY, sl::MEM::GPU);
    zed.retrieveImage(right, sl::VIEW::RIGHT_GRAY, sl::MEM::GPU);

    // C: CUDA preprocess kernel (no PyTorch)
    preprocess_kernel<<<...>>>(left, trt_input);

    // B: TRT C++ infer (no Python wrap)
    context->enqueueV3(stream);
    cudaStreamSynchronize(stream);

    // E: Sparse stereo (CUDA)
    sparse_stereo_kernel<<<...>>>(left, right, keypoints_2d, keypoints_3d, sigma_z);

    // B: SHM write (C++)
    shm_writer.write(keypoints_3d, sigma_z, ...);
}
Success criterion: end-to-end p99 11-13ms
